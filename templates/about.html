About Us

Our project is a basic CRUD application built with Flask, a micro web framework in Python. The app provides RESTful APIs to perform CRUD (Create, Read, Update, and Delete) operations on a database with the following fields: student_id, first_name, last_name, dob (date of birth), and amount_due.

Our team, consisting of Rahul Gupta, Yashika Goel, and Ritvij Sharma, collaborated on this project to build a simple yet functional web app using Flask and other supporting technologies.
